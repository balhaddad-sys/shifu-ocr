## What Was Built This Session

### Architecture Changes (in order)
1. Episodic-first feed (store episodes, extract semantics during replay)
2. 5-checkpoint relay (CP1 specialize → CP2 preprocess → CP3 gate → CP4 handoff → CP5 integrate)
3. Neuroglia system (astrocytes/microglia/OPCs for thermal regulation, healing, neurogenesis)
4. TRN (thalamic reticular nucleus — attentional spotlight between spokes)
5. Fatigue/adenosine model (conviction rests when bridges can't be built)
6. Rust neural core (shifu_neural PyO3 extension — 20x faster wave propagation)
7. Developmental feeding (milk → puree → solid → adult based on vocab size)
8. Wave feeding UI (/dataset command for HuggingFace datasets)
9. Broca's/Wernicke's speaker (trajectory prediction — later removed)
10. Relationship-tagged edges (_co_tags parallel to _co_graph)
11. Motor programs/cerebellum/basal ganglia — then REMOVED in favor of:
12. Pure reactive plasticity (stimulus → field activates → read what vibrates → done)
13. Predictive coding (self predicts, only unexpected changes the self)
14. CPU-timed passive learning (heartbeat pump from Node, 1 BPS)
15. Auto-save every 100 heartbeats
16. Pattern-extracting curriculum

### Key Principles Discovered
- React, don't predict. Change when needed.
- Meaning is not created. It is selected by constraint.
- Think globally. Converge locally.
- Interaction creates meaning, but only a self can recognize it.
- Don't feed an elephant to a baby.
- The node stores the concept. The edge stores the relationship. The goal shapes the path.
